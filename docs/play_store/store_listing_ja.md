# Title
Talkie: AI Language Tutor (会話 & 翻訳)

# Short Description (80 chars)
AIと一緒に実践会話練習。即時翻訳、スピーキング訓練、そして自動復習まで。

# Full Description
スピーキング練習のために設計されたインテリジェントAI語学パートナー、Talkieで眠っていた語学の潜在能力を目覚めさせましょう。

**主な機能:**

🎙️ **リアルタイム発音練習 (Speaking Mode):**
AIがあなたの発音を聞き取り、正確さを評価します。ネイティブの音声を聞いてシャドーイングし、0-100点のリアルタイムフィードバックを受け取りましょう。自分のペースで無制限に反復練習が可能です。

🔍 **スマート検索 & 翻訳:**
世界80以上の言語でテキストを即座に翻訳します。AIが毎日新しい学習文章を推薦し、翻訳された内容は自分だけの学習ノートに即座에保存できます。

📷 **AIスキャン翻訳 (OCR):**
もうテキストを直接入力する必要はありません。カメラで撮影するか、ギャラリーから写真を読み込んでください。Talkieが画像内の文章を即座に認識し、学習言語に翻訳します。気になる文章をタップして発音を聞き、学習リストに追加しましょう。

📖 **体系的な復習:**
一度学んだことを忘れないでください。翻訳内容と学習資料を保存し、体系的な復習システムを通じて長期記憶として定着させましょう。

**🌍 世界80以上の言語をサポート:**
Talkieは現在、世界80以上のすべての言語に対して100%の整合性を保証する翻訳と学習をサポートしています。
- **東アジア:** 韓国語、日本語、中国語 (簡体/繁体)
- **ヨーロッパ:** 英語、スペイン语、フランス语、ドイツ语、イタリア语、ポルトガル语、ロシア语など
- **南アジア:** ヒンディー語、ベンガル語、タミル語など
- **東南アジア:** インドネシア語、ベトナム語、タイ語、フィリピン語など
- **中東およびアフリカ:** アラビア語、トルコ語、スワヒリ語など

**合理的な料金プラン (Flexible Plans):**
- **基本 (Free):** 毎日20回の無料翻訳で始めましょう。翻訳制限に達しても、広告を1つ見ると**即座に10回分チャージ**され、無料で学習を続けられます。
- **プロ (Pro):** アップグレードして広告を削除し、月1,500回 (1日平均50回) の余裕ある翻訳で、途切れることなく学習に没頭しましょう。

**なぜTalkieなのか?**
- **プライバシー重視:** 音声データは安全に処理され、永久に保存されることはありません。
- **オフライン優先:** 学習記録はあなたのデバイスにのみ保存されます。
- **プレミアムデザイン:** 学習のみに集中できる洗練されたグラスモーフィズム3Dインターフェース.

自信を持って話し始めましょう！
