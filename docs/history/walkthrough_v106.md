# 사용자 가이드 및 현지화 최적화 (v106)

앱의 사용성을 높이고 전 세계 사용자가 더 쉽게 시작할 수 있도록 가이드 인터페이스와 현지화 품질을 강화했습니다.

## 주요 변경 사항

### 1. 앱 내 '빠른 시작(Quick Start)' 가이드 도입
- **통합 가이드:** 기존 외부 링크로 연결되던 '사용설명서'를 앱 내 `HelpDialog`의 첫 번째 탭으로 통합했습니다.
- **3단계 핵심 안내:** 새 사용자가 알아야 할 '언어 설정', '기능 흐름', '모드 활용'을 직관적인 아이콘과 설명으로 제안합니다.
- **접근성 개선:** 메뉴(☰)의 '사용설명서'나 '도움말' 클릭 시 즉시 앱 내 가이드가 표시되어 흐름이 끊기지 않습니다.

### 2. 명칭 및 용어 정규화
- **발음 연습 (Speaking Practice):** 기존 '연습' 모드의 명칭을 모드의 성격에 맞게 '발음 연습'으로 변경했습니다. 가이드 내에서도 "텍스트를 보고 직접 발음하며 연습"하는 기능임을 명확히 안내합니다.
- **메뉴 경로 정정:** 가이드 내의 모든 메뉴 명칭을 실제 앱 UI(예: '기기에서 자료 가져오기')와 100% 일치하도록 전수 조사하여 수정했습니다.

### 3. 전 세계 80개 이상의 언어 현지화 업데이트
- **자동 번역 및 배포:** 신규 추가된 '빠른 시작' 문자열과 '발음 연습' 명칭 변경 사항을 80개 이상의 모든 지원 언어(`af`, `en`, `es`, `fr`, `ja` 등)에 자동 반영했습니다.
- **버전 상향:** 모든 변경 사항은 **v2.0.0+106** 버전에 포함되어 안정적으로 적용되었습니다.

### 4. 외부 학습 자료 추출 및 JSON 생성
- **조선일보 연재 기사 활용:** [입에 착착 붙는 일본어](https://www.chosun.com/national/nie/japanese/2026/03/27/S35OFOF65FHSLIC7EBUNDQMQKE/) 기사에서 2026년 3월 27일자 학습 데이터를 추출했습니다.
- **가져오기용 JSON 생성:** 추출한 대화문 3개와 핵심 단어들을 앱의 '기기에서 자료 가져오기' 기능과 호환되는 [japanese_learning_20260327.json](file:///C:/Users/pegas/.gemini/antigravity/brain/d2a765c7-f8f4-47a0-8e3a-b016ef38f91b/japanese_learning_20260327.json) 파일로 생성했습니다.
