# Google Cloud Translation API Key 발급 가이드

Talkie 앱의 번역 기능을 위한 Google Cloud API Key 발급 및 설정 방법입니다.

## 1단계: Google Cloud Console 접속
1.  [Google Cloud Console](https://console.cloud.google.com/)에 접속하고 로그인합니다.
2.  상단 프로젝트 선택 메뉴에서 **[새 프로젝트]**를 클릭합니다.
3.  프로젝트 이름(예: `talkie-translation`)을 입력하고 **[만들기]**를 클릭합니다.

## 2단계: 결제 계정 연결 (필수)
*Google Cloud API를 사용하려면 결제 계정이 연결되어야 합니다. (월 50만 자까지는 무료)*
1.  왼쪽 메뉴에서 **[결제]**로 이동합니다.
2.  **[결제 계정 연결]**을 클릭하고 안내에 따라 신용카드를 등록합니다.
    *   *안심하세요:* 유료 전환 전에 자동 결제되지 않도록 설정할 수 있습니다.

## 3단계: Translation API 사용 설정
1.  상단 검색창에 **"Cloud Translation API"**를 검색합니다.
2.  **[사용]** 버튼을 클릭하여 API를 활성화합니다.

## 4단계: API Key 발급
1.  왼쪽 메뉴에서 **[API 및 서비스] > [사용자 인증 정보]**로 이동합니다.
2.  상단 **[+ 사용자 인증 정보 만들기]**를 클릭하고 **[API 키]**를 선택합니다.
3.  생성된 **API 키**를 복사해 둡니다. (나중에 사용할 것입니다)

## 5단계: API Key 보안 설정 (해킹 방지 - 중요!) ✨
*이 설정을 안 하면 다른 사람이 내 키를 도용해서 요금 폭탄을 맞을 수 있습니다.*

1.  방금 만든 API 키의 오른쪽에 있는 **[키 수정]** (연필 아이콘)을 클릭합니다.
2.  **애플리케이션 제한사항** 설정:
    *   **Android 앱** 선택.
    *   **항목 추가** 클릭.
    *   **패키지 이름**: `com.zeyziyo.talkie` (앱의 패키지명 입력)
    *   **SHA-1 지문**: (터미널에서 `cd android && ./gradlew signingReport` 명령어로 확인 가능)
3.  **API 제한사항** 설정:
    *   **키 제한** 선택.
    *   목록에서 **Cloud Translation API**를 찾아 체크.
4.  **[저장]** 클릭.

---

## 💡 개발용 키 별도 생성 (선택 사항)
위에서 만든 키는 '배포용(Release)'입니다. 개발할 때 컴퓨터(에뮬레이터)에서 테스트하려면 제한이 없는 **'개발용 키'**를 하나 더 만들어서 **개발 중에만 쓰고 배포 전에는 삭제**하거나, `.env` 파일에 넣어 관리하는 것이 좋습니다.
